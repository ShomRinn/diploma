### Current Limitations

| Limitation | Impact | Workaround |
|-----------|--------|------------|
| **No cross-device sync** | Data lost if browser cleared | User must manually export/import |
| **No backup** | Chat history lost on device failure | None |
| **Single device only** | Can't access from phone/tablet | None |
| **Storage quota** | localStorage limited to ~5-10MB | Auto-cleanup (50 chat limit) |
| **No encryption** | Data readable by any script | Browser security model only |
